# Nerf-MA40-Dart-Counter
Nerf blaster visual dart counter and clip capacity detector.

Parts List:
  - One Arduino compatible board of some kind
  - Two Hall Effect sensors
  - One I2C OLED Screen
  - One microswitch (perferably with a wire)
